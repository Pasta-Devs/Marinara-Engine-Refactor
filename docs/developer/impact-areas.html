Shared Change Risk

engine/contracts

Review schemas, DTO consumers, shared API adapters, and any persisted records.

engine/generation-core

Review chat, roleplay, game, agents, prompt preview, and replay behavior.

src/shared/api

Review all features calling the adapter and the matching Tauri command DTO.

src/shared/components

Review every feature surface using the component, including mobile layout.

If the impact area cannot be named clearly, the change is probably in the wrong owner or too broad for one commit.
