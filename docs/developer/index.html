How this repo is put together and how to work in it without widening impact.

Marinara is a Tauri desktop app. React owns the user interface, the TypeScript engine owns product behavior, and Rust owns privileged local capabilities such as storage, filesystem access, provider transport, secrets, and integrations.

Rules That Keep Changes Small

One owner

Every behavior belongs to React UI, the TypeScript engine, or Rust capability code. Do not duplicate ownership across layers.

Mode separation

Chat, roleplay, and game are sibling product modes. They share lower-layer primitives, not each other's orchestration.

Explicit imports

Import owner modules and public feature APIs. Avoid deep imports into another feature's private files.

Root fixes

Fix broken contracts or owners. Do not add fake success paths, broad fallbacks, or UI-only patches.

Before a change: identify the owner, expected impact area, dependent callers, and checks. If a shared layer changes, verify every mode or feature that depends on that layer.
